# Web.UI/public/content - AI Context

## Purpose

This folder contains all static content assets for the MarbleLife web application, including fonts, images, stylesheets, and JavaScript files.

## Structure

- **fonts/**: Custom web fonts (Kievit font family)
- **images/**: All image assets (logos, icons, flags, social media icons)
- **scripts/**: JavaScript libraries and application scripts
- **styles/**: CSS stylesheets for application styling

## For AI Agents & Human Developers

Static content folder organized by asset type:

### Fonts (`fonts/`)
- Custom typography (Kievit font family)
- Web font files (.woff, .ttf, .eot, .svg)
- Font face definitions in CSS

### Images (`images/`)
- Application logos and branding
- Icons and UI elements
- Flag images for internationalization
- Social media icons
- Layout-specific images

### Scripts (`scripts/`)
- Third-party JavaScript libraries
- Vendor scripts (jQuery, Angular, Bootstrap, etc.)
- Custom application scripts
- Utility functions

### Styles (`styles/`)
- Global CSS stylesheets
- Component-specific styles
- Responsive design rules
- Theme customizations

### Best Practices:
- Optimize images for web (compression, appropriate formats)
- Use CDN for common libraries when possible
- Minify CSS and JavaScript for production
- Use image sprites for small icons
- Implement lazy loading for images
- Version static assets for cache busting
- Organize by functional area within each category
