# Web.UI/core/filters - AI Context

## Purpose

This folder contains AngularJS filters for the **core** module.

## Contents

Filters files that implement:
- Business logic and data operations (if services)
- View logic and user interactions (if controllers)
- Reusable UI components (if directives)
- HTML templates and markup (if views)
- Data transformation (if filters)

## For AI Agents & Human Developers

**Filters** in this folder follow AngularJS 1.x patterns:
- Use dependency injection for services
- Implement proper error handling
- Follow established naming conventions
- Maintain separation of concerns
- Test components appropriately
