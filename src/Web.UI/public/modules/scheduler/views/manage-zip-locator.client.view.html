Steps to Upload Zip Code File:

1. Click on Download File and make changes into your downloaded GeoCode file.

2. This file as 2 data tabs – a ZIP-to-COUNTY assignment tab and a FRANCHISE-to-COUNTY tab

3. Periodically we may need to add a NEW zip to the COUNTY, in which case that line ID should be set to 0 which will signify that this is a NEW and ADDED line.

4. Periodically we may need to change FRANCHISE assignments for the COUNTY. Remember each franchise is a STATE-METRO AREA name.

5. PHONE number is updated on the FRANCHISE DASHBOARD via the TRANSFER PHONE check box on the list of phone numbers in the first column

6. EMAIL is updated via the MARKETING EMAIL box in the DASHBOARD

7. Those records that you want to edit, change there column IsUpdated to 1 and records that you want to delete, change column IsDeleted to 1.

8. If there is any change in ZipCode need to make IsUpdated to 1 For its corresponding County in County Tab.
